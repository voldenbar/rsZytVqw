# 项目简介

## 核心功能
- 基于 YOLOv8 的航拍视角目标检测系统，可识别 10 类目标：行人、人群、自行车、汽车、面包车、卡车、三轮车、带篷三轮车、公交车、摩托车
- 支持图片、视频及摄像头三种输入方式进行检测
- 提供图形用户界面，实时显示目标位置、目标总数和置信度等信息
- 支持自定义置信度和 IOU 阈值
- 可选择显示检测标签和原图
- 支持批量检测并查看所有结果
- 支持检测结果保存

## 适用场景
- 航拍图像分析
- 交通监控与车辆统计
- 人群密度分析
- 安防监控系统
- 自动驾驶辅助系统
- 城市规划与研究

## 技术特点
- 基于 Python 3.9 开发，使用 OpenCV、PyQt5、Ultralytics 和 Torch 1.9 等技术栈
- 提供完整源码、8000 张标注数据集、训练好的模型及详细运行教程
- 配置成功环境后可直接使用，无需额外训练
- 软件兼容 PyCharm+Anaconda 或 VSCode+Anaconda 开发环境
